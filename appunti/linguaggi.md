## JS JAVASCRIPT
javascript è l'unico linguaggio di front end che il browser è in grado di capire.

## TS TYPESCRIPT
è usato per il front end quindi il programmatore non ha boisogno di imparare linguaggi diversi essendo uguale a JS.

## LE ARCHITETTURE DEI PROXCESSORI

## CISC

## RISC

## PYTHON

si riduce la sintassi ma l'ordine degli spazi diventa importanti.
Ogni cosa in python è un oggetto, perfino una variabile.

## NODE JS
