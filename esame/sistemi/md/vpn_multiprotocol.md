# VPN BGP/MPLS (MultiProtocol Lable Switching)
Sostituisce gli IP delle reti private con delle lable,
il quale vengono gestite da router speciali.

Si opera nel DATA LINK, il traffico e' piu' veloce e si riesce a comunicare con protocolli diversi.

Si usano 3 router per farla:
- CE (Customers Edge Router):
- PE (Provided Edge Router):
- P (Provider Core Router):

I CE sono esterni,
I PE sono in mezzo,
I P sono interni alla rete.

L'MPLS e' tra il DATA LINK e il NETWORK LAYER.
